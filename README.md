## 프로젝트 목표
* 기면증, 뇌졸중 등으로 힘들어하는 환자들이 인적이 드문 곳에서 쓰러졌을 때, 구조대에게 곧바로 환자의 위치를 알려 3분 내로 출동하게 한다.
* 동시에 환자가 쓰러졌을 때, 부모님/가족에게 문자로 쓰러짐을 알려준다.
